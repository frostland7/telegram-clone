# Telegram Clone Project

## Overview
This project is a clone of the popular messaging application, Telegram. It aims to provide real-time communication features, including text messaging, voice calls, and file sharing.

## Features
- User Authentication
- Real-time messaging
- Group chats
- Voice and video calls
- File sharing
- Push notifications

## Technologies Used
- React.js for the front-end
- Node.js and Express for the back-end
- MongoDB as the database

## Setup Instructions

### Prerequisites
- Node.js and npm installed
- MongoDB installed and running

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/frostland7/telegram-clone.git
   cd telegram-clone
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables in a `.env` file. You can refer to the `.env.example` for the required variables.
4. Run the application:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to see the application.
